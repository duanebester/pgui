# PGUI

A GUI to query & manage postgres databases.

Written in [GPUI](https://gpui.rs) and [GPUI Component](https://github.com/longbridge/gpui-component)

### Saved Connections

Connections will be saved to a sqlite db file in `~/.pgui/connections.db`
